🎯 我們要解決什麼痛點？

👨‍🏫 使用對象：教練

  • 整合教學前置準備： 系統化管理學生資訊與教學進度。
  • 主動釋出空檔： 教練發布「可教學時間與地點」，讓學生主動預約選購。
  • 社群需求爬蟲系統： 平台自動去社群爬取學生的「求教練文」，精準推播配對！

🏂 使用對象：學生

  • 全方位推薦指南： 不再盲人摸象！平台智慧推薦雪場、雪道、住宿與裝備。
  • 專屬滑雪日誌： 紀錄自己的滑行歷程，留下真實評價，造福下一位雪友。
  • 串聯社群分享： 完美整合滑行紀錄與心得分享，讓滑雪成為社群亮點！

🏗️ 我們的技術火力 (Tech Stack)

  • ⚛️ 前端： Next.js + Tailwind CSS (支援 PWA 離線快取)
  • 🗄️ 資料庫： Supabase (開源的 Firebase 替代方案)
  • 🚀 伺服器： Zeabur 官方贊助極速節點！
  • ⚖️ 版權保護： AGPLv3 開源授權，保護心血

👑 推舉核心幹部與小組長

🚀 專案架構小組 (階段性)

專案草創期的先鋒部隊！

  • 負責敲定資料庫架構 (Schema)
  • 決定網站路由與基礎框架設定
  • 階段任務完成後即解散回歸各組

⭐ 三大任務小組「組長」

不是主管，是跨組溝通的橋樑！

  • 條件：常在線上、回覆快、熱心
  • 任務：協調認領進度、主持小組討論
  • 無階級壓力，未來定期輪替制
