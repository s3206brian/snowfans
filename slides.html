❓ 為什麼需要 SnowFans？

現有痛點

  • 資訊散落各地、農場文太多
  • 教練平台抽成過高、不透明
  • 裝備交易充滿詐騙
  • 歐美雪場缺乏中文攻略

我們的願景

  • 非營利、無抽成、絕對透明
  • 維基百科式共筆協作
  • 建立健康互助的雪友生態圈
  • 成為大家職涯履歷上的亮點！

🏗️ 我們的技術火力 (Tech Stack)

  • ⚛️ 前端： Next.js + Tailwind CSS (支援 PWA)
  • 🗄️ 資料庫： Supabase (開源的 Firebase 替代方案)
  • 🚀 伺服器： Zeabur 官方贊助極速節點！
  • ⚖️ 版權保護： AGPLv3 開源授權，保護心血

👑 推舉初代小組長 (Team Leads)

  • 任務： 不是當主管！而是當「跨組溝通橋樑」
  • 條件： 常在線上、回覆快、熱心整理重點
  • 機制： 定期輪替制，沒有上下級壓力
  • 收穫： 寫在履歷上超加分的專案管理經驗！
