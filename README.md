> **「找教練怕踩雷？買裝備怕被騙？歐美雪場沒資訊？」**
> 受夠了散落各地的滑雪農場文，我們決定自己開源寫一個！
> 
> 本專案由工程師兼自由滑雪教練發起，致力於打造一個屬於全體雪友的**非營利、無抽成、資訊透明**的 PWA (漸進式網頁應用) 資源整合平台。

## 🌟 核心功能與願景

打破現有滑雪資訊散落、商業抽成過高的痛點，我們採取 **開源協作 (Open Source)** 與 **維基共筆 (Wiki-based)** 精神：

1. 🤝 **透明化教練媒合：** 建立實名與證照審核的教練名錄，學生與教練直接聯繫，平台 **0 抽成**。
2. ♻️ **二手雪具交易板：** 專屬雪友的跳蚤市場與安心代購資訊。
3. 🏠 **住宿與避坑指南：** 嚴選高 CP 值住宿（主推背包客棧）及雪具店真實評價。
4. 🌍 **歐美夢幻雪場拓荒指南 (獨家)：** 打破日韓紅海，專攻 Epic / Ikon Pass 跨洲神票攻略、阿爾卑斯山與北美雪場交通指南。

## 🏗️ 技術架構 (Tech Stack)

我們採用現代化全端架構 (Modern Stack)，為未來擴充 iOS/Android App 鋪路，並確保極佳的開發者體驗：

- **前端框架:** [Next.js (App Router)](https://nextjs.org/) + [React](https://reactdev.com/)
- **UI & 樣式:** [Tailwind CSS](https://tailwindcss.com/) + [shadcn/ui](https://ui.shadcn.com/)
- **後端與資料庫:** [Supabase](https://supabase.com/) (PostgreSQL + Auth + Storage)
- **部署與託管:** [Zeabur](https://zeabur.com/) (支援 PR 預覽環境 CI/CD)
- **PWA 支援:** 支援雪場無網/弱網離線快取，並可加入手機主畫面。
