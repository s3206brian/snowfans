⚛️ 前端與 PWA

Next.js (React) + Tailwind CSS。支援離線快取，讓雪友在無網的雪山環境也能查看攻略。

🗄️ 後端與資料庫

Supabase (PostgreSQL)。強大的開源 Firebase 替代方案，完美支援共筆系統與權限控管。

🚀 部署與託管

Zeabur。台灣團隊打造的神級 PaaS，提供極速 CI/CD 與 PR 預覽環境，降低溝通成本。
